Task operations
~~~~~~~~~~~~~~~

Use the following operations in this section to perform tasks for importing and exporting
zones for a Rackspace Cloud account.

Imports:

	- :ref:`Create a zone import <POST_importZone_v2__account_id__zones_tasks_imports_zones>`
	- :ref:`List zone imports <GET_listZoneImports_v2__account_id__zones_tasks_imports_zones>`
	- :ref:`List a zone import <GET_listZoneImport_v2__account_id__zones_tasks_imports__zone_id__zones>`
	- :ref:`Delete a zone import <DELETE_deleteZoneImport_v2__account_id__zones_tasks_imports__zone_id__zones>`

Exports:
	- :ref:`Create a zone export <POST_exportZone_v2__account_id__zones__uuid_id__tasks_export_zones>`
	- :ref:`List zone exports <GET_listZoneExports_v2__account_id__zones_tasks_exports_zones>`
	- :ref:`List a zone export <GET_listZoneExport_v2__account_id__zones_tasks_exports__uuid_id__zones>`
	- :ref:`Delete a zone export <DELETE_deleteZoneExport_v2__account_id__zones_tasks_exports__uuid_id__zones>`
	- :ref:`List an exported zone <GET_listExportedZone_v2__account_id__zones_tasks_exports__uuid_id__export_zones>`
